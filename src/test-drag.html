Image Drag & Drop Test

Test Instructions:

  1. Open the main application at http://localhost:5174/LandingPage/
  2. Insert an image using the image button in the toolbar
  3. Click on the image to select it (blue border should appear)
  4. Look for the green circular handle at the top-left corner
  5. Click and drag the green handle to move the image
  6. Try to drop it between paragraphs or at different positions

Expected Behaviors:

✓ Green drag handle appears when image is selected
✓ Cursor changes to "grab" over the green handle
✓ While dragging, cursor changes to "grabbing"
✓ A drop indicator line appears showing where image will be placed
✓ Image moves to new position when dropped
✓ Text reflows around the image based on wrap setting

Common Issues & Solutions:

Issue: No drag handle visible
Solution: Click on the image first to select it
Issue: Image doesn't move when dragging
Solution: Make sure you're dragging the GREEN handle, not the image itself
Issue: Can't drop image
Solution: Look for the drop indicator line and drop there

Debug Console

Open browser DevTools (F12) and check the Console tab for any errors related to:

  • NodeViewWrapper
  • ReactNodeViewRenderer
  • data-drag-handle
  • TipTap drag events
